# Loading Dots Animation

Projeto desenvolvido com HTML e CSS que apresenta uma animação de carregamento com múltiplos pontos. A animação utiliza variáveis CSS e `@keyframes` para criar um efeito dinâmico e moderno.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3

## ✨ Funcionalidades

- Loading animado
- Múltiplos pontos em movimento
- Animação com CSS puro
- Uso de variáveis CSS
- Sem JavaScript

## 📂 Estrutura do Projeto

```bash
Loading-Dots-Animation/
│
├── index.html
├── css/
│   └── style.css
└── README.md
```

## 🎯 Como Executar

1. Clone ou baixe o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` no navegador.

## 📚 Conceitos Praticados

- CSS Animations
- Keyframes
- Variáveis CSS
- Position
- Delays de animação
- Componentes de carregamento
